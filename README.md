In this project we intend to find some relationships between influenza vaccine coverage and other demographic features, including education, gender, population, etc. in the United States in recent years. If possible, we also plan to dig deeper into the impact of COVID-19 on the flu vaccination coverage.

Link to the book: https://etevanss.github.io/VaccinationCoverageDemographics/
